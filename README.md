# Welcome to Covid-19 App

## Prerequisite

- NVM
- Node JS 14.5
- Docker
- React

## How to start the development

Without docker

```bash
npm install

npm run start
```

or using docker

```bash
# to docker up
docker-compose up -d

# to docker down
docker-compose down
```
